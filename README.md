## Objectives
- Enhance data security by encrypting transaction metadata using Advanced Encryption Standard (AES).
- Hide encrypted data within digital images using Adaptive Pixel Value Difference (PVD) steganography, preventing visibility of sensitive information.
- Introduce randomized embedding using the AES key to make the embedding locations unpredictable and resistant to steganalysis attacks.
- Improve imperceptibility by adapting the embedding capacity based on image texture (smooth vs edge regions).
- Ensure reliable extraction and decryption of hidden metadata at the receiver side without loss of information.
- Evaluate performance using metrics such as PSNR, MSE, payload capacity, and extraction accuracy.
